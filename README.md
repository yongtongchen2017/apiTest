# autotest-byPytest
## 运行命令（带allure报告）：
###1，pytest -v --alluredir ./temp --baseUrl https://api2.mubu.com
###2，allure generate ./temp -o ./allure-report --clean
###3，allure open ./allure-report

### 注：命令行执行前先确保python虚拟环境正确，即工程下又venv目录且安装了相关插件。
